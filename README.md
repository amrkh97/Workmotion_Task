## Getting Started

### 1. Package the Application:

- ```mvn clean package```

### 2. Build the docker images:  

- ```docker-compose  build```

### 3. Run the docker image:

- ```docker-compose  up```

### 4. Check the openApi/Swagger documentation:

[Swagger Link](http://localhost:80/swagger-ui/index.html)

### 5. Enjoy!
